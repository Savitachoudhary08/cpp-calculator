# C++ Calculator

A moderately advanced calculator built using C++.  
It supports both basic and scientific operations with Degree and Radian modes.

---

## 🔧 Features

### Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

### Advanced Operations
- Power (^)
- Square Root (r)
- Exponential (e)
- Natural Log (ln)
- Log base 10 (log)

### Trigonometric Functions
- Sine (s)
- Cosine (c)
- Tangent (t)
- Inverse Sine (S)
- Inverse Cosine (C)
- Inverse Tangent (T)

### Modes
- Degree Mode
- Radian Mode

### Other Functionalities
- Continuous calculations (e.g., 3 + 5 + 7 + ...)
- Error handling (like divide by zero)

---

## ▶️ How to Use

1. Run the program
2. Enter numbers and operators as shown in the menu
3. Press `=` to get the result
4. Restart the program for a new calculation

---
